GE实习之Module WS

发表于 2018-04-22 | 分类于 GE实习 ， 数据准备

通用电气公司（General Electric Company，简称GE），是世界上最大的提供技术和服务业务的跨国公司。通过自己的不懈努力，成为了GE的一名数据分析实习生，借博客这个平台记录自己的实习过程，希望不仅可以帮助他人，也可以总结自己，获得更大的提升。这篇博客主要讲解实习过程中为Module WS.xlsx文件录入数据的详细细节，以及在DSS学习中学到的Acronyms。

数据填写规则

  • WS Validation列
    原有45台 ESN（Electronic Serial Number，电子序列号），在WorkScope中查看每台ESN的pdf文件，在GE的EWT网址中搜索ESN号，点击Generate Customer Report 生成的便是该ESN的pdf文件。填写WS Validation列，填写的规则是：
    1.没有的模块填写NA
    2.HMU模块，根据查找到的description，填写的是Visual Inspection 或 Visually Inspect
    3.M15-17 Fan Case模块，pdf中的M15 Fan Case等同于该模块，pdf中的Fan-Stator M17 level 则在WS Validation列填入Fan-Stator M17 I 等
    4.M23、M24、M31、M32模块属于M40模块，若有level值则填入对应的level值，若没有level值或M23、M24、M31、M32模块，则在WS Validation 列填入NA.M40 II(ESN中M40模块的level值)
  • Sub Module WS列和WS Validation列
    新加18台ESN，在WorkScope里查看每台ESN的pdf文件，填写Sub Module WS列和WS Validation列，填写规则是：
    1.Sub Module列每个模块，若有对应的level值，则在Sub Module WS列中填入相应的level值（1即为I，2即为II……）
    2.Sub Module列的模块，若没有，则Sub Module WS空着，即不填也不写WS
    3.若没有HMU模块，则在Sub Module WS中空着，在WS Validation中填入对应的description，例如：Visual Inspection 或 Visually Inspect
    4.M23、M24、M31、M32模块属于M40模块，若有level值则填入对应的level值，若没有level值或M23、M24、M31、M32模块，则在WS Validation 列填入NA.M40 II(ESN中M40模块的level值)
  • Cost level列与Comments列
    Cost level列与Comments列的填写规则是：
    1.复制整列Sub Module WS列内容至Cost level列，若Sub Module WS为空，则Cost level列填SA
    2.Filter出Sub Module列的M10 Engine模块，将Cost level全部改为SA
    3.Filter出Sub Module列的M71模块，若Sub Module WS为IV，则Cost level也为IV，若Sub Module WS为其它，则Cost level改为II
    4.Filter出Sub Module列的M58模块，查看每个ESN的pdf文件，若M58的Comments为Route TCF to vendor for repair，则将该描述添加进Comments列，且若Sub Module WS为II，则将Cost level改为III
  • Block列与Part Number列
    在GE的TEXL GE MRO Data Collaboration网站上找到新加的18台ESN的.zip文件，下载并解压。Block列与Part Number列的填写规则是：
    1.在每个ESN的.zip文件中找到Configuration Report R00.xlsx文件并打开，例如查到的HPT S1N 的PN号为2084M68G11
    2.在该ESN的.zip文件中找到Life Limited Parts Report R00.xlsx文件并打开，查看Description列中的SEAL-INTERSTAGE HPT ROTOR IN对应的PN号（以Input为准），该PN号与Block的对应关系如下表所示：
    PN为2505M72P01的，在Block列天上EIS Old，Part Number列标注上2505M72P01，若PN为其它，则在Block列标注PN值对应的Block值即可，Part Number列不标注。（询问Mentor后，Mentor与技术人员沟通后做出的该规定方法）

  • EIS, Block 2012, Block 2014, Block 2015 definition
Block Definition
EIS At least one of the following is true: HPT S1N: G11 OR G11/G15, HPT S2N: G07 OR G07/G13, TCF Cooling holes = no, OGV<>P, Shroud = P10 OR P16.
Block 2012 All of the following must be true: HPT S1N: G15 and up, HPT S2N: G13 and up, TCF Cooling holes = yes, OGV leaf seal = P, Shroud = P15(or any Combination of the following: Shroud = P17 or Spoolie = Yes or Combustor Inner Liner = G05 or G06 or HPT S1 Blade = P09 as long as Block 2014 or Block 2015 definition is not met)
Block 2014 All of the following must be true: HPT S1N: G15 and up, HPT S2N: G13 and up, TCF Cooling holes = yes, OGV leaf seal = P, Shroud = P15, Spoolie migration fix = yes, HPT S1B = P09, Combustor Inner Liner = G07(or any Combination of the following: Shroud = P17 or Combustor Dome = 2110M68G06 or 2256M54G03 as long as Block 2015 definition is not met)
Block 2015 HPT S1N: G15 and up, HPT S2N: G13 and up, TCF Cooling holes = yes, OGV leaf seal = P, Shroud = P17, Spoolie migration fix = yes, HPT S1B = P09, Combustor Inner Liner = G07, Combustor Dome = 2110M68G06 or 2256M54G03
